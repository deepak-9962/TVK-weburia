How to Use

  • Enter username and password for the new admin user
  • Click "Generate Hash & SQL" to create the password hash
  • Copy the SQL query and run it in Supabase SQL Editor
  • The admin user will be created immediately
  • Use the username and password to login at admin-login.html

Security Best Practices

  • Use strong passwords (min 12 characters, mixed case, numbers, symbols)
  • Never share admin credentials
  • Change default passwords immediately
  • Regularly rotate admin passwords
  • Keep track of all admin users
