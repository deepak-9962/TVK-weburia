ஊழியர் மேலாண்மை (Employee Management)

Add, Edit, and Manage Employee Accounts

முக்கிய குறிப்பு (Important Note)

  1. Employees must be created in Supabase Authentication first
  2. Go to: Supabase Dashboard → Authentication → Users → Add User
  3. Enter email and password for each employee
  4. Then click "Add New Employee" here to create employee record
  5. Use suggested password format: TVK@2025!EmpXXX
